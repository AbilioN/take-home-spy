--------------------------------------------------
PROJECT GOALS
--------------------------------------------------

1) This is a user-only mobile app pretending to be a "Cat Spotter."
2) The app should:
   - Show a list of random cats using TheCatAPI
   - Ask for device location
   - Send location to backend (POST /locations)
   - Optionally fetch last known location for this user (GET /users/:id/last-location)
3) The Admin and full location history are NOT part of this app.

--------------------------------------------------
USER FLOW
--------------------------------------------------

CatHomeScreen:

1) On load:
   - Request device location permission
   - Get current location
   - Fetch list of cats from TheCatAPI
2) Display:
   - Cat images
   - Current latitude & longitude (optional)
3) Buttons:
   - "Send My Location" → POST /locations
   - "View Last Location" → GET /users/:id/last-location → navigates to LastLocationScreen

LastLocationScreen:

- Fetch GET /users/:id/last-location
- Display latitude, longitude, timestamp
